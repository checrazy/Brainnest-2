Questions

  1. What are the main differences between external, internal, and inline CSS?
    The main difference between inline CSS and external CSS is that inline CSS is processed faster as it only requires the browser to download 1 file while using external CSS will require downloading HTML and CSS files separately.
  2. What is the syntax for class and ID selectors?
    For the class is .[name of class] and for id is #[name of id]
  3. How would you apply a single rule to two different selectors?
    By using comma ex. div, p { color: #f00; }
  4. Given an element that has an id of title and a class of primary, how would you use both attributes for a single rule?
    #title.primary.
  5. What does the descendant combinator do?
    A combinator is something that explains the relationship between the selectors.
  6. Between a rule that uses one class selector and a rule that uses three type selectors, which rule has the higher specificity?
    The one that uses three type selectors.
  7. From inside to outside, what is the order of box-model properties?
    Content, Padding, Border, Margin
  8. What does the box-sizing CSS property do?
    The box-sizing property sets how the total width and height of an element is calculated.
  9. What is the difference between the standard and alternative box model?
    Padding and border don't increase the width
  10. Would you use margin or padding to create more space between 2 elements?
    Margin
  11. Would you use margin or padding to create more space between the contents of an element and its border?
    Padding.
  12. Would you use margin or padding if you wanted two elements to overlap each other?
    Negative Margin
  13. What is the difference between a block element and an inline element?
    A block-level element always starts on a new line and an inline element does not start on a new line.
  14. What is the difference between an inline element and an inline-block element?
    It is the parent of the nested element.
  15. What relationship do two elements have if they are at the same level of nesting?
    You can set the size of the inline-block element.
  16. Is an h1 block or inline?
    h1 to h6 is a block
  17. Is button block or inline?
    Button is Inline
  18. Is div block or inline?
    div is a block element
  19. Is span block or inline?
    The span element is an inline container
  20. What's the difference between a flex container and a flex item?
    Flex Container is the parent element while Flex Item represents the children.
  21. How do you create a flex item?
    By creating a flex container as his parent.
  22. What are the 3 values defined in the shorthand flex property?
    The 3 values are flex-grow flex-shrink and flex-basis
  23. How do you make flex items arrange themselves vertically instead of horizontally?
    align-items.
  24. What is the difference between justify-content and align-items?
    justify-content — controls alignment of all items on the main axis. align-items — controls alignment of all items on the cross axis.
  25. How do you use flexbox to completely center a div inside a flex container?
    By adding the display: flex; and to center out item horizontally we use the justify-content: center;
  26. What's the difference between justify-content: space-between and justify-content: space-around?
    Space-between is used for displaying flex items with space between the lines and space-around is used for displaying flex items with space between, before and after the lines.
